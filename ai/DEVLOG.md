## 2026.03.22

### 기능 개선
- **커뮤니케이션 코치 채팅 통합**: 별도 사이드 패널 제거, 메인 채팅에서 자연어로 질문하면 오케스트레이터가 자동 감지하여 처리
  - 채팅 입력창 placeholder에 커뮤니케이션 관련 예시 힌트 추가
- **Kiwi 형태소 분석 적용**: RAG 검색 전 질문에서 핵심 명사/동사 추출 → 쿼리 확장으로 검색 정확도 향상
- **말투 감지 로직 개선**: "연차"처럼 짧고 모호한 입력을 반말로 오해하던 문제 수정
  - 반말 감지 조건 강화: 반말 표현이 명확히 있을 때만 반말 모드 전환
  - 시스템 프롬프트 기본값을 "편안한 톤" → "정중한 존댓말"로 명시

### 버그 수정
- **ChromaDB HNSW 인덱스 손상 문제**: OneDrive 동기화가 바이너리 파일을 덮어써 인덱스가 깨지는 현상
  - 해결: ChromaDB 저장 경로를 OneDrive 외부(`~/withbuddy_chroma_db`)로 이동

## 2026.03.21

### Streamlit 데모 페이지 구현 (`streamlit_app.py`)
FastAPI 서버 없이 AI 기능을 직접 테스트할 수 있는 Streamlit 버전 구현

**구현 페이지 (11개)**
| 페이지 | 주요 기능 |
|--------|-----------|
| 💬 채팅 | RAG 기반 질의응답, 오케스트레이터 연동, 대화 요약/초기화 |
| 📊 리포트 | 주간 온보딩 리포트 생성 |
| 🔍 담당자 추천 | 질문 입력 또는 칩 선택으로 담당자 자동 추천 |
| ✅ 체크리스트 | 부서별 온보딩 체크리스트 생성 |
| 🗺️ 로드맵 | 온보딩 진행 로드맵 |
| 📋 마이버디 | 메모, 빠른 질문, 태스크 관리 |
| 📄 문서함 | 사내 문서 목록 조회 및 내용 보기 |
| 🎉 온보딩 | 환영 레터 및 팀 소개 카드 (프리보딩) |
| 👤 내 프로필 | 사용자 프로필 입력/수정 |
| 👔 사수 페이지 | 미답변 질문 확인 및 답변 입력 |
| 🔧 관리자 | 회사 정보 등록, 팀원 관리, 리포트 발송 |

**설계 특이사항**
- FastAPI 라우터 없이 Python 모듈 직접 import로 동작 (빠른 프로토타이핑 목적)
- 별도 배치 파일(`start_streamlit.bat`)로 실행
