# WithBuddy Frontend

> React + Vite 기반 프론트엔드

## 🧭 프로젝트 표준

| 항목 | 값 |
|------|-----|
| 디렉토리 | `frontend/` |
| 프로젝트명 | `withbuddy-frontend` |
| 언어 | JavaScript (ES6+) |
| 기본 포트 | 5173 |

## 🛠 기술 스택

- React 18
- Vite
- React Router
- Context API (JWT는 localStorage 저장)
- Tailwind CSS
- Axios

## 🚀 시작하기

```bash
cd frontend
npm install
npm run dev
# http://localhost:5173
```

## 환경변수

```bash
VITE_API_BASE_URL=http://localhost:8080/api
```
